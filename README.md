## 📌 项目背景

<details>
<summary><b>点击查看项目初衷</b></summary>

由于教委平台存在诸多技术问题：
- 平台优化缺失
- 无移动端自适应
- APP短信验证码服务不稳定
- 功能维护不足
- 安全防护薄弱

## 🚀 项目开发历程

| 阶段 | 方案 | 结果 |
|------|------|------|
| 第一阶段 | 使用Python浏览器自动化适配手机UI | ❌ 功能性问题失败 |
| 第二阶段 | 本地CSS持续修改实现自适应 | ❌ 环境问题失败 |
| 第三阶段 | 油猴脚本修改网站CSS | ❌ 浏览器私密性限制 |
| 第四阶段 | 模拟PC端访问的内置浏览器 | ⚠️ 效果一般 |

> 由于《网络安全法》等法规限制，项目最终停止开发，但仍提供现有解决方案。
